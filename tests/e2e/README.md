# E2E testlar

Ishga tushirish:

1. Ilovani alohida terminalda ishga tushiring:
   - `npm run dev`
2. Testlarni ishga tushiring:
   - `npm run test:e2e`

Ixtiyoriy:

- Agar ilova boshqa portda ishlasa:
  - `E2E_BASE_URL=http://localhost:3001 npm run test:e2e`

Eslatma:

- Hozircha testlar skeleton/smoke ko'rinishida.
- Keyingi bosqichda seed + login fixture bilan to'liq business-flow testlar qo'shiladi.
